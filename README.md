# WALL-E

WALL-E is a discord bot, [get started](https://discord.com/oauth2/authorize?client_id=1217855376960258090).

## Features

1. Get latest currency exchange rate from [OpenExchangeRates](https://openexchangerates.org/)
2. Shorten URL from [is.gd](https://is.gd)
3. Generate QR code with given text (or url)
4. Random pick one from two given options
5. Random pick one from multi options
6. Random pick a number by given range
7. Transfer time by given timezones

## User Privacy

- This bot will and never collect any user data.
- All interaction will not be stored.

## Dependencies

- [discord.js](https://github.com/discordjs/discord.js)
- [qrcode](https://github.com/soldair/node-qrcode)
- [axios](https://github.com/axios/axios)
- [mersenne-twister](https://github.com/boo1ean/mersenne-twister)
- [city-timezones](https://github.com/kevinroberts/city-timezones)
